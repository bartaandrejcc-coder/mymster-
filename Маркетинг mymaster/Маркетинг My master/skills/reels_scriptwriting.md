# SKILL: Сценарії Reels (Instagram / TikTok)

## Чому Reels критично важливі для СТО
- Алгоритм IG/TikTok дає масовий охоплення Reels (більше за пости в 5-10×)
- Авто-нiшa візуальна: процес фарбування / поліровки сам по собі гіпнотичний
- Можна показати "до/після" як ніде

## Формули хуків (перші 3 секунди)

### Формула 1: Питання + цифра
> "Знаєш скільки коштує НЕ зробити поліровку перед продажем авто?"

### Формула 2: Парадокс
> "Цей лак гірший за заводський. І ви не помітите різниці."

### Формула 3: Несподіване твердження
> "Локальне фарбування дешевше, ніж ви думаєте."

### Формула 4: Раптовий показ результату
> [Кадр шикарного результату] → "Ось як це виглядало 3 дні тому"

### Формула 5: Зворотнє
> "3 ознаки що НЕ варто йти в цей сервіс"

## 8 шаблонів-сценаріїв (60 сек)

### Шаблон 1: До/Після з музикою

```
0-3: ХУК — кадр пошкодженого авто крупним планом
4-10: "Привіз клієнт після ДТП. Думав — здавати в металобрухт."
11-30: Швидкі склейки процесу (5-8 кадрів по 2-3 сек)
31-45: Голос за кадром: "3 дні роботи. Підбір кольору 1:1. Без зварки..."
46-55: Кадр готового авто + крутиться навколо
56-60: "Запис на огляд — у Direct. My Master, Мукачево"
```

### Шаблон 2: Поради майстра (3 пункти)

```
0-3: "Ці 3 помилки коштують вам грошей. Я механік 12 років. Слухайте."
4-20: Помилка 1 + пояснення + кадр з прикладом
21-37: Помилка 2 + ...
38-54: Помилка 3 + ...
55-60: "Більше порад — підписуйтесь. My Master"
```

### Шаблон 3: Експеримент

```
0-3: "Ми взяли 2 однакові деталі. На одну — наш лак, на другу — гаражний."
4-50: Процес + порівняння + видумане катування (вода, удар, час)
51-60: Результат — "ось чому ми не економимо на матеріалах"
```

### Шаблон 4: Сторителлінг клієнта

```
0-3: Клієнт у кадрі: "Шукав СТО 3 місяці."
4-30: Він розказує проблему / шлях
31-55: Показ роботи / результату
56-60: Клієнт: "Тепер тільки My Master"
```

### Шаблон 5: Закадровий розбір

```
0-3: Фото проблеми + текст-питання: "Скільки коштує це виправити?"
4-50: Поетапний розбір процесу і ціни
51-60: "Запис на безкоштовний огляд. Точну вартість — за 30 хв"
```

### Шаблон 6: "Не платіть це"

```
0-3: "За цю роботу хочуть 8000 грн? Не платіть."
4-50: Розбір що насправді потрібно і скільки повинно коштувати
51-60: "Чесні ціни — у нас. My Master"
```

### Шаблон 7: Зйомка процесу satisfying

```
0-3: Хук — крапля води на щойно полірованому крилі
4-50: Етапи поліровки крупним планом, повільно
51-60: "Поліровка від 1500 грн. Запис в Direct"
```

### Шаблон 8: Сезонний

```
0-3: "Весна. 3 речі, які треба зробити з авто ЗАРАЗ."
4-50: 3 поради + наочно для кожного
51-60: "Усе разом — від 800 грн. Запис у Direct"
```

## Технічні правила

### Звук
- Завжди тренди (актуальні)
- НЕ використовуй приглушений / тихий звук
- Голос диктора краще тільки в Stories, у Reels — частіше музика + субтитри

### Субтитри
- ОБОВ'ЯЗКОВО (80% людей дивляться без звуку)
- Великий шрифт (можна читати в авто)
- Контрастний фон під текстом

### Тривалість
- Найкраще 21-30 сек (швидко, але встигаємо передати ідею)
- 60 сек — для серйозного контенту (кейс, тривалий процес)
- Не більше 90 сек (алгоритм не любить)

### Перший кадр (cover)
- Має бути візуально захоплюючий
- Текст-хук на cover
- "Заборонені" cover: розмиті, темні, статичні

### Hashtag-стратегія для Reels
- 8-10 хештегів максимум
- 3 трендових (за датою)
- 3 нішевих
- 2-3 локальних
