# SKILL: SEO-аудит сайту

## Коли застосовувати
Раз на квартал або при будь-якій помітній зміні сайту.

## Чек-лист SEO-аудиту My Master (mymastermyk.com.ua)

### 1. Технічна частина

- [ ] **PageSpeed Insights** — оцінка ≥80 (mobile + desktop)
  → https://pagespeed.web.dev/
- [ ] **HTTPS** — сертифікат валідний
- [ ] **Mobile-friendly** — тест Google
- [ ] **Sitemap.xml** — присутній і доданий у Search Console
- [ ] **Robots.txt** — не блокує важливі сторінки
- [ ] **Canonical URL** — встановлені для дублів
- [ ] **404 сторінка** — налаштована, з посиланнями назад

### 2. Структура сайту

Обов'язкові сторінки:
- [ ] Головна
- [ ] Кузовний ремонт (окремо)
- [ ] Фарбування авто (окремо)
- [ ] Поліровка (окремо)
- [ ] Кондиціонери (окремо)
- [ ] Ремонт після ДТП (окремо)
- [ ] Про нас / Команда
- [ ] Контакти + карта
- [ ] Блог
- [ ] Кейси / Портфоліо (До/Після)
- [ ] Відгуки

### 3. On-page SEO (кожна сторінка)

| Елемент | Вимога |
|---------|--------|
| Title | 50-60 символів, з основним ключем + локацією |
| Meta Description | 150-160 символів, з ключем + CTA |
| H1 | Один на сторінці, з основним ключем |
| H2-H6 | Структурована ієрархія |
| Alt у картинок | Описовий, з ключем |
| URL | Короткий, читабельний (/farbuvannya-avto, не /page?id=23) |
| Внутрішні посилання | 3-5 на кожній сторінці |
| Контент | Мін 800 слів для посадкових |

### 4. Локальне SEO (Google Business Profile)

- [ ] Профіль створений і верифікований
- [ ] Назва: My Master
- [ ] Категорія: Auto body shop / Автосервіс
- [ ] Адреса точна
- [ ] Телефон з кодом міста
- [ ] Сайт додано
- [ ] Робочі години правильні (з поправкою на свята)
- [ ] Опис 750 символів з ключами
- [ ] Послуги додані (всі)
- [ ] Фото — мін 30 шт (зовні, всередині, до/після, команда)
- [ ] Перші 100 відгуків — план до Q3
- [ ] Пости — раз на тиждень
- [ ] FAQ — заповнено

### 5. Ключові слова — пріоритети

**Транзакційні (high intent):**
- кузовний ремонт мукачево
- фарбування авто мукачево
- сто мукачево
- автосервіс мукачево
- поліровка авто мукачево

**Інформаційні:**
- скільки коштує пофарбувати авто
- як підготувати авто до фарбування
- що таке локальний ремонт кузова

### 6. Контент-план блогу (для SEO)

1. "Скільки коштує пофарбувати авто у 2026? Розбір за регіонами"
2. "Локальний ремонт vs повне фарбування: коли що краще"
3. "Чому колір не сходиться: 5 причин і як уникнути"
4. "Поліровка фар своїми руками: чому це поганий ідея"
5. "Як вибрати СТО для кузовного ремонту: 7 ознак"
6. "Антикор для авто у Закарпатті: що, коли, скільки"
7. "Кондиціонер в авто: коли діагностика і скільки коштує"

Цільова частота: 1 стаття/тиждень, мін 1500 слів кожна.

### 7. Що відстежувати

- Google Search Console (покази, кліки, CTR, позиції)
- Google Analytics (трафік, поведінка, конверсії)
- Позиції за топ-30 ключами (Serpstat / Semrush / руками)
- Кількість backlinks (Ahrefs / Mass)
