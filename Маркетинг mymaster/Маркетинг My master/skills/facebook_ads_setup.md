# SKILL: Налаштування Facebook/Instagram Ads

## Структура рекламного кабінету

```
БМ My Master
├── Кампанія: Conversions — Заявки на ремонт
│   ├── Ad Set 1: Холодна — гео Мукачево + інтерес "авто"
│   ├── Ad Set 2: Холодна — Look-alike 1%
│   └── Ad Set 3: Теплі — відвідувачі сайту 30/90 днів
├── Кампанія: Reach — Впізнаваність бренду
├── Кампанія: Lead Generation (форма в Meta)
└── Кампанія: Engagement (бустер постів)
```

## Налаштування аудиторій

### Холодні аудиторії (Cold)

**A1 — Власники авто Мукачево**
- Гео: Мукачево + 30 км
- Стать: чол + жін (60/40)
- Вік: 25-55
- Інтереси: Автомобілі, BMW, Audi, Volkswagen, Skoda, Mazda
- Поведінка: подорожі, час за кермом

**A2 — Авто-публіки регіону**
- Гео: Закарпатська область
- Demo: 25-55, чол
- Інтереси: Автомобілі + Авторемонт + Ремонт авто
- Включити: Engaged shoppers

**A3 — Lookalike 1%**
- Джерело: список телефонів клієнтів (мін 100)
- Гео: Закарпаття
- Розмір: 1%

### Теплі (Warm)

**W1 — Відвідувачі сайту**
- Pixel ивент: Page View, останні 30 / 90 / 180 днів

**W2 — Взаємодія з IG/FB**
- Дії: переглянули профіль, надіслали повідомлення, відреагували на пост
- Період: 90 днів

**W3 — Відео-видивлювачі**
- Подивилися 50%+ нашого відео за 90 днів

### Гарячі (Hot)

**H1 — Залишали заявку, але не прийшли**
- Custom Audience з CRM

**H2 — Підписники, які давно не реагували**
- Підписалися 6+ міс тому, не реагували останній місяць

## Креативи — формати

### 1. Static Image (1:1 і 4:5)
- Фото "До/Після" (2 кадри)
- Інфографіка з цінами
- Фото команди + ціна послуги

### 2. Carousel (3-5 слайдів)
- Слайд 1: Хук
- Слайди 2-4: Процес роботи
- Слайд 5: CTA + контакти

### 3. Video (9:16 для Stories/Reels, 1:1 для Feed)
- Тривалість: 6-15 сек для Stories, 15-60 для Feed
- Перші 3 сек критичні — захоплюємо увагу
- Субтитри обов'язково (більшість дивляться без звуку)

### 4. Collection (для мобільних)
- Cover video / image
- 4-8 продуктових фото знизу (послуги з цінами)
