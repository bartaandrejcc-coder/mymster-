# Шаблон рекламного оголошення

## Для Meta (Facebook/Instagram Ads)

### Основний текст (Primary Text) — до 125 символів видно без "more"

```
[Хук — біль або питання]
[Коротка пропозиція + 2-3 переваги]
[CTA + локація]
```

**Приклад:**
```
Подряпина від паркінгу? Виправимо за 1 день. 
My Master, Мукачево. Гарантія 12 міс. Безкоштовний огляд.
👉 Запис у Direct
```

### Заголовок (Headline) — до 27 символів

Приклади:
- Кузовний ремонт Мукачево
- Поліровка авто від 1500₴
- Фарбування без зсувів
- Безкоштовний огляд

### Опис (Description) — до 27 символів

Приклади:
- Гарантія 12 місяців
- Підбір кольору 1:1
- Записуйтесь зараз

### CTA-кнопка

Варіанти:
- Send Message / Написати
- Learn More / Дізнатись
- Get Quote / Дізнатись ціну
- Call Now / Зателефонувати
